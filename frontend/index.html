✈ Itinerary

Sistema de gestión de itinerarios de viaje · Universidad Central · 2026

Aplicación web para planificar itinerarios de viaje entre aeropuertos nacionales e internacionales, desarrollada como proyecto de Ingeniería de Software II.

El sistema está construido sobre una arquitectura de microservicios, separando la gestión de aeropuertos y la gestión de itinerarios en servicios independientes que se comunican entre sí mediante APIs REST.
